The Process

Research

First task was to re-familarize myself with the angular framework by using the Angular JS Docs. The documentation provided a refresher to architecture, components and of course the use of HttpClient to fetch API's.

Implimentation

I began the project by building a basic application that utilized a home, about , and contact page. Each having functionallity that re-familiarize me with the framework befor moving toward api integration.

I used the first public park API available which was National Park Service. The documetation wasn't the most comprehensive and honesty had missing documentation that had been previously available for earlier versions which i found trough research via stack overflow.

Angular Google maps (AGM) was an excellent map API with good documentation and plenty of github uses that i was able to impliment and make use of.

Style

using the w3schools.com documentation i was able to style the application using some of the latest UI/UX style available. I also made use of photoshop to quickly render a custom background and icons.
